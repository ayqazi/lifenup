# Explanatory Page — Parenting Plan: Purpose and Limits

This parenting plan is included because you asked for the issue to be noted and appended, not finally resolved now.

## Purpose

- to record present intentions
- to reduce future ambiguity
- to identify issues for later agreement, mediation, arbitration, or family court if needed
- to keep the children’s welfare central

## Limits

- It is not a binding child arrangements order.
- It does not fix an exact pattern of care.
- The court will decide child arrangements by reference to the children’s welfare at the relevant time if a dispute arises.

## Why include it anyway

It can still help by:

- showing both parents anticipated the issue
- recording an intention for both parents to remain involved
- reducing later claims that one parent was intended to be excluded
- giving a starting framework for later negotiation
